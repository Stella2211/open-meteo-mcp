# プライバシーを考慮したOpenMeteo MCPサーバー

## 概要
このプロジェクトは、プライバシーを考慮したOpenMeteoのMCPサーバーを実装するものです。
機能は非常にシンプルなもので、特定の地点の今後7日間の予報を取得するのみです。
実装の背景：大手AIプロバイダーが提供しているAIのほとんどは、ユーザーとの会話内容を学習に使用しています。ここに自宅の緯度経度を入力することは、プライバシー上の懸念があります。
しかし、利便性の観点から、「自宅の天気予報を教えて」のようにAIに指示したいです。
そのため、このMCPサーバーでは、AIが直接緯度経度に触れる機会を極限まで低減させ、プライバシー上のリスクを低減することを目的としています。

## 機能
- OpenMeteoのAPIを用いて、特定の地点の今後7日間の予報を取得する
- 地点情報はユーザーのホームディレクトリ/.config/stella2211-open-meteo-mcp/locations.jsonに保存する
- AIは地点の追加・削除、地点名一覧の取得・当該地点の天気予報の取得が可能
- 地点の追加時以外、その場所の緯度と経度がAIには伝わらない

## 要件
- npxで起動可能
- claudeで利用可能な形式のMCPサーバー
- win,mac,linux対応
- プライバシーに配慮
